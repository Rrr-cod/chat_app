# Chat Application

A real-time chat application built using MERN Stack and Socket.io.

## Features
- User Authentication
- Real-time Messaging
- Chat Rooms
- Responsive UI

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- Socket.io

## Installation

Client:
npm install
npm start

Server:
npm install
npm run server
